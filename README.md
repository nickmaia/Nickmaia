### Projetos

**[MACES](https://github.com/nickmaia/MACES)**
Análise estrutural de moléculas via espectrometria de massas — MLP + algoritmo genético. TCC de Engenharia Química.

**[Modelitx](https://github.com/nickmaia/Modelitx)**
Gerador de funções a partir de dados reais para ensino de Cálculo. Bolsa FAPEMIG. → [demo](https://modelitx.vercel.app/)

**[conditional-molecular-design-ssvae](https://github.com/nickmaia/conditional-molecular-design-ssvae)**
Semi-Supervised VAE para predição de propriedades e geração de moléculas.

**[raven](https://github.com/nickmaia/raven)**
Reprodução em escala reduzida de experimentos de pesquisa.
